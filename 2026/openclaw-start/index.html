安装与升级（Windows / macOS）

一键安装

请确保您的系统已连接互联网，并拥有相应的执行权限。

为什么不推荐一键安装？

  1. 无法指定安装位置
  2. 依赖互联网，不支持内网
  3. 脚本的行为难以掌控，有安全风险

手动安装 Node.js

阿里云镜像下载地址：https://registry.npmmirror.com/binary.html?path=node/v24.14.0/

建议下载长期支持版本（LTS，大版本号为偶数）以获得更稳定的运行环境。

Windows 下载 node-v24.14.0-x64.msi，macOS 下载 node-v24.14.0.pkg（不区分CPU）。

https://registry.npmmirror.com/-/binary/git-for-windows/v2.53.0.windows.2/Git-2.53.0.2-64-bit.exe

人格调教

官方中文提示词：https://docs.openclaw.ai/zh-CN/reference/templates/AGENTS

  • AGENTS：入口
  • BOOTSTRAP：初始化引导
  • HEARTBEAT：心跳
  • IDENTITY：龙虾的个人信息
  • SOUL：龙虾的性格
  • TOOLS：工具参数、环境信息
  • USER：用户的个人信息
  • AI 会自主决策创建更多文件

记忆

记忆是智能体工作空间中的纯 Markdown 文件。这些文件是唯一的事实来源；模型只”记住”写入磁盘的内容。

短期记忆：memory/YYYY-MM-DD.md

长期记忆：MEMORY.md

心跳与定时任务

心跳

让 AI 主动干活，主动发消息

最佳实践：跳过夜间，节省 token，避免打扰。

示例：你每天都会在随机时间主动给我发随机的 3-6 条消息，只能在我醒着的时间，即 7 点到 22 点之间发，可以但不限于聊天气、最近发生的事情、科技新闻等，我会回复你，是否对你的话题感兴趣。你需要通过我的反应不断地学习我的喜好，从而更加准确地发起我感兴趣的话题，你可以把这项规则写进心跳里

示例：帮我监控 https://item.jd.com/100235893165.html 的价格，降价了提醒我

示例：在心跳中检查邮箱中的邮件，总结成摘要发给我

定时任务

指定时间干活

示例：每天早上 7:20，如果郑州市金水区当天有雨，提醒我，否则不提醒

安全管控

  • 运行环境隔离
    • 必须使用干净、隔离的操作系统环境，如龙虾迷你主机、虚拟机或独立的云服务器实例，避免影响主系统安全。
  • Skill 查毒
    • 安装前扫描技能，检测恶意软件、间谍软件、挖矿程序及恶意代码模式，识别数据外泄、系统篡改、后门与混淆技术。
    • skillhub install skill-scanner
  • 定期「换锁」
    • 本地网关认证——别让任何人都能连上你的 AI
    • 定期更换网关 token、大模型 key
  • 防止提示词注入——别让 AI 被”催眠”

云端一键部署

本地客户端

  • 腾讯 QClaw：本地客户端，免费，每天 4000 万 token
  • 智谱 AutoClaw：本地客户端，模型收费，自定义模型不收费，有较强的浏览器操作能力
  • 讯飞 Loomy：本地客户端，需邀请码，底层不是 OpenClaw

云端

大厂有钱就是任性：

  • 阿里 JVSClaw：云端 + 手机 APP，免费，需邀请码，相当于一台装了原版 OpenClaw 的云电脑，能装软件、执行命令，自由度极高，token 无限，用完可以找客服免费续

收费门槛较高：

  • 字节 ArkClaw：云端，收费每月 200
  • 月之暗面 KimiClaw：云端，收费每月 199
  • 飞书 OpenClaw：云端，免费每天 100 万 token，收费每月 69 但额度极少（大概不到 1000 万 token）

定价高到离谱：

  • 网易 OpenClaw 云电脑：原版 OpenClaw，贵，一小时 2 元，模型调用还要另算，每轮对话 1~2 元，关机数据不保留

移动端

打通米家生态：

  • 小米 MiClaw：手机 APP，免费，需邀请码，底层不是 OpenClaw，只支持小米 17 系列

总结

  • 夯：阿里 JVSClaw、小米 MiClaw
  • 顶级：智谱 AutoClaw、腾讯 QClaw
  • 人上人：字节 ArkClaw、月之暗面 KimiClaw
  • NPC：讯飞 Loomy、飞书 OpenClaw
  • 拉完了：网易 OpenClaw 云电脑
